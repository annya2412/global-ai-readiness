# Global AI Readiness Dashboard

Interactive dashboard analyzing AI readiness across countries.

## Run the project:
1. `pip install -r requirements.txt`
2. `python data_collection.py`
3. `streamlit run app.py`

## Features:
- World map of AI readiness
- Top 10 countries ranking
- GDP vs Readiness correlation
- Country comparison

Created with Python, Streamlit, Plotly
